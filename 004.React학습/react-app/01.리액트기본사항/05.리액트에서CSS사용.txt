05. 리액트에서 CSS 사용하기

1. 컴포넌트를 구성하는 리액트에서는
일반적으로 CSS파일을 import하여 사용함

2. import한 CSS파일을 html페이지 직접
CSS구역으로 배포되며 이것은 인터널 방식이다

3. CSS공통파일 합치기
-> 문법: @import() 
-> 경로도 현재폴더가 같다면 현재 경로사용
-> style구역을 따로 만들어서 인터널로 삽입함
-> import 순서는 동일함(위에 넣으면 순서대로)

4. 배경이미지 경로
-> public 폴더에 있는 이미지는
CSS위치로부터 상대적으로 잡아준다
예) background-image:url(../../public/images/aa.jpg)

5. 개별요소 인라인 CSS넣기
jsx문법 style속성에 객체형식으로 넣는다
-> 외부에 스타일객체를 만들고 그 객체를 호출해도됨!
예) <div style={{color:'red'}}
